
# About

此项目是 vue + vuex + axios 构建的后台管理系统，所有的数据都是从服务器实时获取的真实数据，具有真实的注册、登陆、管理数据、权限验证等功能。


# 说明





## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 


## 项目运行


```
git clone https://github.com/bailicangdu/vue2-manage  

cd vue2-manage  

npm install 

npm run dev 

访问: http://localhost:8002

```
